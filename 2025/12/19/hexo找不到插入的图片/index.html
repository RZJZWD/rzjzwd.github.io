hexo找不到插入的图片

发表于 2025-12-19 更新于 2026-02-12 分类于 hexo
解决hexo无法插入图片的问题

打开资源文件夹

hexo为管理文章图片提供了一个同名文件夹的功能，在配置文件中找到post_asset_folder: true并开启，这样在每次创建新文章时会自动创建一个与文章同名的文件夹，并且从草稿到发布时也会自动拷贝文件夹

安装插件

安装插件npm install hexo-asset-image --save根据仓库所写

和第一步是一样的

还是有问题

如果在完成以上两步后还是没有图片，那么就需要在hexo根文件路径下的public文件夹下，根据文章日期找到文章文件夹，查看下面究竟有没有图片
也可以在本地查看时通过浏览器开发者模式查看是否有图片被正确复制
如果正确复制到public下的同名文件夹但是图片还是没有，那么就刷新hexo的缓存，重新生成，这个方法对于已发布文章再添加图片但是不刷新有用
